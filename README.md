## How to run

1. Lock node version

```
nvm install && nvm use
```

2. Install dependencies

```
npm install
``` 

3. Run in local environment

```
npm start
```

- Visit at: http://localhost:3000/
- Inspect
- `toggle device toolbar`

![inspect.png](./docs/toolbar.png)


4. Run tests

```
npm test
```
